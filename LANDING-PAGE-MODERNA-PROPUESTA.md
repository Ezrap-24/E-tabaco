## 2. Paleta de Colores Moderna

### Colores Primarios
```
- Fondo: #FFFFFF (blanco) o #F8F9FA (gris muy claro)
- Texto: #1A1A1A (casi negro)
- Acento Principal: #FF6B35 (naranja vivo) o #E63946 (rojo enérgico)
- Acento Secundario: #2A9D8F (verde agua) o #264653 (azul oscuro)
```

### Uso
```
- CTAs (botones): #FF6B35 (naranja)
- Hover: #E55100 (naranja más oscuro)
- Éxito/Stock: #2A9D8F (verde)
- Alerta/Oferta: #E63946 (rojo)
- Información: #264653 (azul)
```

**Referencia:** Estilo similar a Airbnb, Stripe, Figma (moderno, limpio, energético).

## 3. Tipografía

```
Títulos: Poppins o Inter (Sans-serif, bold)
         Tamaños: H1=48px, H2=36px, H3=28px

Body: Inter o Roboto (Sans-serif, regular)
      Tamaño: 16px
      Line height: 1.6

CTAs: Poppins Bold, 16px, MAYÚSCULAS

Peso visual: Mucho en titulares, poco en descripciones
```

---

## 4. Experiencia de Usuario (UX)

### Mobile First
```
- Diseño pensado para celular primero
- Menú hamburguesa en móvil
- Imágenes optimizadas (no más de 100KB)
- Touch targets grandes (botones mín 48px)
```

### Velocidad
```
- PageSpeed > 90 (Google Lighthouse)
- Imágenes lazy-loaded
- CSS/JS minificado
- CDN para assets
```

### Conversión
```
- CTAs aparecen cada 3-4 secciones
- Carrito siempre visible (sticky header)
- Checkout en máx 3 pasos
- Forms simples (mín. campos)
```

---

## 5. Diferenciación vs. Competencia

### vs. Smoke Jokers
```
✗ Smoke Jokers: Gris y sobrio, enfoque premium
✓ Nuestra landing: Colores vivos, energía joven
         → Más atractiva para público 18-40 años
```

### vs. La Tabaquería
```
✗ La Tabaquería: Magenta + naranja caótico, poco info
✓ Nuestra landing: Orden visual, secciones claras, propuesta de valor
         → Más profesional, mejor conversión
```
