## 🎨 Cambios en Paleta de Colores

### v1.0 (Simple)
| Color | HEX | Rol |
|-------|-----|-----|
| Negro | #1A1A1A | Principal |
| Oro | #C4A17A | Acento |
| Crema | #F4EDE4 | Secundario |
| Borgoña | #5C2E2E | CTA |
| Gris | #4A4A4A | Textos |

**Problema:** Negro + oro = elegante pero impersonal, no dice "tabaco mayorista"

| Color | HEX | Rol | Significado |
|-------|-----|-----|-------------|
| Verde Tabaco | #30483A | Principal | Herencia tabacalera |
| Tabaco Claro | #C8B08A | Acento | Calidez profesional |
| Marfil Vintage | #EFE6D6 | Secundario | Elegancia clásica |
| Gris Pizarra | #3A3A3A | Textos | Firmeza, autoridad |
| Bronce Envejecido | #8B7355 | CTA | Valor, experiencia |

**Ventaja:** Verde + tabaco claro = reconocible como industria tabacos, más sofisticado

---

## 🔤 Cambios en Tipografía

### v1.0
- **Títulos:** Georgia (serif genérica)
- **Body:** Segoe UI (sans-serif moderna)
- **Resultado:** Clásico pero no muy memorable

### v2.0 (Tu idea)
- **Títulos:** Playfair Display (serif elegante, histórica)
- **Body:** Lora (serif con carácter, más sofisticada)
- **Resultado:** Más clásico, más memorable, heritage visible

**Por qué:** Playfair + Lora transmiten "tradición establecida" mejor que Georgia + Segoe
