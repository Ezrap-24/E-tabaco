### PROMPT COMPLETO V2.0

```
Diseña una landing page profesional para "PURO TABACO", una mayorista premium 
de tabaco especializada en conectar empresas confiables con marcas líderes.

CONTEXTO HISTÓRICO Y NARRATIVO:
- Puro Tabaco conecta el legado tabacalero de Londres (Est. 1902) con el 
  comercio mayorista moderno chileno.
- NO es una tienda retail de lujos, ES una mayorista B2B seria y profesional.
- Mensaje clave: "Tradición que mueve negocios"
- Audiencia: Gerentes de negocio, dueños de distribuidoras, mayoristas

IDENTIDAD DE MARCA:
- Nombre: PURO TABACO
- Subtítulo: Mayorista de Tabaco Premium
- Dominio: purotabaco.cl
- Posicionamiento: Premium B2B Mayorista
- Público: Negocios confiables, distribuidores, empresas
- Tono: Profesional, cercano, directo, con autoridad histórica
- Narrativa: Herencia + Eficiencia = Negocios sólidos

PALETA DE COLORES (NUEVA - MÁS ROBUSTA):
- Color primario: Verde Tabaco #30483A (headers, fondos)
- Color secundario: Tabaco Claro #C8B08A (acentos, bordes)
- Color terciario: Marfil Vintage #EFE6D6 (fondos secundarios)
- Color oscuro: Gris Pizarra #3A3A3A (textos principales)
- Color de énfasis: Bronce Envejecido #8B7355 (botones CTA)

TIPOGRAFÍA:
- Títulos: Playfair Display (serif clásica, elegante)
- Body: Lora (serif con carácter, legible)
- Buttons: Uppercase, letter-spacing prominente
- Tono tipográfico: Clásico pero moderno, heritage pero profesional

LOGO Y MARCAJE:
- Logo escudo clásico con "EST. 1902" (autoridad histórica)
- Puede ser estilo vintage pero limpio, no ornamentado
- Logo tipo: "PURO TABACO" en Playfair Display
- Línea descriptiva: "Mayorista de Tabaco Premium"

ESTRUCTURA DE SECCIONES:

1. HEADER/NAVEGACIÓN:
   - Fondo: Verde tabaco #30483A
   - Logo a la izquierda con "Est. 1902"
   - Navegación horizontal: INICIO | CATÁLOGO | MARCAS | CONTACTO | [ACCESO MAYORISTAS]
   - Línea divisora dorada/tabaco claro en parte inferior
   - Texto claro (crema), profesional

2. HERO SECTION (Narrativa Principal):
   - Fondo: Imagen vintage de época (Big Ben + carro de tabaco, estilo sepia/clásico)
   - O fondo sólido verde tabaco con elemento decorativo
   - Titular GRANDE: "TRADICIÓN QUE MUEVE NEGOCIOS" 
     (en Playfair Display, color tabaco claro #C8B08A)
   - Subtítulo: "Desde 1902, conectando empresas confiables 
     con tabaco premium de calidad mundial"
   - Dos botones CTA:
     * "SOLICITAR CATÁLOGO" (Bronce #8B7355, texto crema)
     * "VER MARCAS LÍDERES" (Secundario, borde tabaco claro)
   - Espacios generosos, elegancia profesional

3. SECCIÓN "SOBRE PURO TABACO":
   - Fondo: Blanco o crema suave
   - Texto narrativo corto:
     "Mayorista de tabaco premium desde 1902. Conectamos empresas confiables 
      con las marcas líderes del mundo. Stock permanente, precios mayoristas 
      competitivos, despachos rápidos a todo Chile."
   - Pueden haber 3-4 valores visualizados:
     * Herencia: Icono de reloj/edificio histórico
     * Confianza: Icono de manos/handshake
     * Calidad: Icono de estrella/seal
     * Eficiencia: Icono de check/velocidad
   - Cada valor con descripción corta

4. SECCIÓN "MARCAS LÍDERES":
   - Fondo: Verde tabaco #30483A (MÁS OSCURO que el header para contraste)
   - Titular blanco: "DISTRIBUIDORES CERTIFICADOS DE MARCAS LÍDERES"
   - Logos de marcas en grid (4-5 logos):
     Dunhill, Mac Baren, Davidoff, Marlboro, Pueblo
   - Logos en color claro (crema/white)
   - Texto: "Somos distribuidores oficiales de las mejores marcas de tabaco 
     mundial. Garantía de autenticidad y frescura en cada producto."

5. SECCIÓN "VENTAJAS MAYORISTA":
   - Fondo: Marfil #EFE6D6
   - Línea divisora tabaco claro
   - 4-5 cards en grid mostrando beneficios:
     
     * STOCK PERMANENTE
       Inventario actualizado de todas nuestras marcas
     
     * PRECIOS MAYORISTAS
       Márgenes competitivos para tu negocio
     
     * DESPACHOS RÁPIDOS
       Entregas a todo Chile en 2-3 días
     
     * ATENCIÓN PERSONALIZADA
       Equipo especializado dedicado a tu negocio
     
     * GARANTÍA DE AUTENTICIDAD
       Productos 100% originales, sellos fiscales válidos

   - Cada card: Fondo crema, borde izquierdo tabaco claro, icono pequeño
   - Tipografía clara, sin florituras

6. SECCIÓN "CÓMO FUNCIONA" (Si aplica):
   - 4 pasos visuales:
     1. SOLICITAR → Pide acceso mayorista o catálogo
     2. REVISAR → Explora nuestras marcas y precios
     3. ENCARGAR → Realiza tu pedido con condiciones mayoristas
     4. RECIBIR → Despacho rápido a tu negocio
   - Fondo blanco, conectados con líneas tabaco claro
   - Cada paso con número grande + icono + descripción

7. SECCIÓN "¿QUIÉNES COMPRAN CON NOSOTROS?":
   - Fondo: Crema suave
   - Texto: "Empresas de todos los tamaños confían en nosotros"
   - Ejemplos/tipos de clientes (SIN nombres reales):
     ✓ Distribuidoras de bebidas y cigarrería
     ✓ Locales de venta al detalle
     ✓ Negocios de gastronomía y esparcimiento
     ✓ Mayoristas y revendedores

8. SECCIÓN "CÓMO COMPRAR EN PURO TABACO":
   - Fondo: Blanco o crema suave
   - Titular: "3 Pasos fáciles para tu pedido"
   - Tres pasos visuales:
     1. SELECCIONA → Explora nuestro catálogo de marcas
     2. COMPRA → Carrito simple, pago seguro
     3. RECIBE → Despacho rápido a tu negocio
   - Cada paso con número grande + descripción
   - Énfasis en: mayorista, rápido, confiable

9. SECCIÓN CTA FINAL (GRANDE):
   - Fondo: Verde tabaco #30483A
   - Titular en blanco: "¿LISTO PARA LLEVAR TU NEGOCIO AL SIGUIENTE NIVEL?"
   - Descripción: "Somos mayoristas con más de 120 años de experiencia. 
     Pregúntanos sobre volúmenes, condiciones especiales y programas."
   - Botón grande: "SOLICITAR ACCESO MAYORISTA"
   - Debajo: WhatsApp visible +56 9 1234 5678 (como alternativa)
   - Opcional: Email de contacto

10. FOOTER:
   - Fondo: Verde tabaco muy oscuro (casi negro) #1A1A1A
   - Texto: Crema/blanco
   - Acentos: Tabaco claro #C8B08A
   - Links: Sobre Nosotros, Catálogo, Marcas, Contacto, Términos, Privacidad
   - Redes sociales: Instagram, Facebook, LinkedIn
   - Copyright: "© 2026 Puro Tabaco — Mayorista de Tabaco Premium. Est. 1902"
   - Dirección opcional: Si aplica

ESTILO VISUAL GENERAL:
- Minimalismo profesional: espacios generosos, Sin saturación
- Colores nobles y sólidos: Verde tabaco, tabaco claro, bronce, crema
- Fotografía estilo vintage/clásica: Tabaco enrollado, cajas, elementos históricos
- Tipografía clásica (Playfair) + moderna (Lora) = Heritage + Profesional
- Sensación de: Autoridad, Confianza, Eficiencia, Solidez
- Responsive design: Perfecto en móvil, tablet, desktop
- Sombras SUTILÍSIMAS: Casi no se notan, muy elegante
- Bordes redondeados: Máximo 2px (casi rectos, muy clásico)
- Líneas decorativas: Solo tabaco claro, nunca en exceso

REQUISITOS TÉCNICOS:
- Formato: Desktop-first pero responsive
- Colores EXACTOS: #30483A, #C8B08A, #EFE6D6, #3A3A3A, #8B7355
- Tipografías: Playfair Display (títulos), Lora (body)
- Sin animaciones excesivas, pero sí hover effects suaves en botones
- Menú fácil de navegar en móvil (hamburguesa si es necesario)
- WhatsApp visible como opción de contacto
- Enfoque B2B: Lenguaje de negocios, no de retail

TONE OF VOICE EN TEXTOS:
- Profesional pero accesible
- Directo sin ser frío
- Que respete la historia sin ser arcaico
- Que inspire CONFIANZA en negocios
- Ejemplos:
  ✓ "Distribuidores certificados de las mejores marcas"
  ✓ "Precios competitivos para tu negocio"
  ✓ "Más de 120 años de experiencia en la industria"
  ✗ NO: "Comprá el mejor tabaco del mundo" (retail, no profesional)

INSPIRACIÓN VISUAL:
- Heritage minimalista (tipo Hermès, Loro Piana)
- B2B profesional (tiendas de vinos premium, distribuidoras)
- Elegancia vintage (1900-1920 Londres)
- Paleta: Verdes oscuros + bronces + crema (profesional, no lujo excesivo)

Por favor diseña una landing que sea:
✓ Seria, profesional y confiable (B2B)
✓ Visualmente coherente con autoridad histórica
✓ Que transmita competencia en el sector
✓ Fácil de navegar para empresarios ocupados
✓ Responsive para todos los dispositivos
✓ Que inspire "Sé que puedo confiar en este mayorista"
✓ CON PALETA NUEVA: Verde + Bronce + Crema (NO negro + oro)
```

## 🎯 Instrucciones de uso

1. **Abre Canva Design:**
   - Ve a https://claude.ai/design

2. **Copia el prompt completo** (arriba) en el chat

3. **Espera a que genere opciones** — recibirás varios diseños candidatos

4. **Selecciona el que más te guste:**
   - Revisa que tenga los colores correctos (verde, tabaco claro, crema)
   - Verifica que la narrativa sea "Tradición que mueve negocios"
   - Asegúrate que se vea profesional B2B (no retail)

5. **Descarga o guarda:**
   - PNG para vista rápida
   - PPTX para editar después
   - O guarda como proyecto en Canva

## 💡 Si necesitas ajustes

### Más minimalista:
```
Simplifica a 6 secciones: Hero, Sobre nosotros, Marcas, Ventajas, CTA, Footer.
Mantén la narrativa pero reduce elementos visuales.
```

### Más visual/fotos:
```
Agrega más fotografía: tabaco enrollado, cajas vintage, carros históricos.
Estilo moodboard con referencias visuales de época.
```

### Enfoque mayorista más fuerte:
```
Enfatiza: "Somos mayoristas, no retail"
Agrega sección "Volúmenes y condiciones" o "Acceso mayorista"
Destaca precios competitivos, despachos rápidos
```

### Mobile-first:
```
Prioriza diseño para pantallas pequeñas (320px).
Menú en hamburguesa, botones grandes, espacios claros.
Texto claro y fácil de leer en móvil.
```
