# Glosario — e-Tabaco

## Términos del proyecto

| Término | Significado |
|---------|-------------|
| **e-Tabaco** | Nombre del proyecto de e-commerce para la tabaquera |
| **catálogo** | Página que lista todos los productos disponibles |
| **ficha de producto** | Página detalle de un producto individual |
| **checkout** | Flujo de finalización de compra (datos + pago) |
| **gate de edad** | Pantalla/modal de validación de mayoría de edad al entrar al sitio |
| **SKU** | Código único de identificación de cada producto |
| **local** | La tienda física donde se va a hacer la sesión de fotos |

## Stack técnico

| Término | Significado |
|---------|-------------|
| **Django** | Framework web Python usado para el backend |
| **PostgreSQL** | Base de datos principal |
| **Bootstrap** | Framework CSS para el diseño responsive |
| **MVP** | Minimum Viable Product — primera versión funcional mínima |

## Convenciones

| Elemento | Convención |
|----------|------------|
| Imágenes de productos | `marca-tipo-peso.jpg` (ej: `marlboro-rojo-20u.jpg`) |
| Carpeta de imágenes | `/media/products/` |
| Carrito | Implementado con sesiones de Django (sin login requerido) |
