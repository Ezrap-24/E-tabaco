# Contexto: Sesión fotográfica de productos

**Fecha programada:** 2026-03-31 (mañana)
**Lugar:** El local físico de la tabaquera

## Qué fotografiar

- Todos los paquetes de tabaco disponibles
- Vista frontal, lateral y detalle de cada producto
- Presentaciones individuales y de bulto (si aplica)
- Marcas y variantes disponibles

## Tips para la sesión

- Usar fondo neutro (blanco o gris claro) para fotos de producto
- Luz natural o artificial uniforme, sin sombras duras
- Fotografiar en horizontal y vertical para tener opciones
- Incluir escala visual si los tamaños varían mucho

## Convención de nombres

```
marca-tipo-presentacion.jpg
```
Ejemplos:
- `marlboro-rojo-20u.jpg`
- `camel-azul-10u.jpg`
- `particulares-virginia-paquete.jpg`

## Destino de archivos

```
/media/products/
```

## Qué registrar también (además de las fotos)

Para armar el catálogo necesitás anotar por cada producto:
- Nombre completo de la marca
- Tipo / variante (rojo, azul, mentol, etc.)
- Presentación (10 unidades, 20 unidades, cajilla, etc.)
- Precio de venta al público
- Precio mayorista (si aplica)
- Descripción breve

## Checklist antes de ir

- [ ] Cámara / celular con buena resolución cargado
- [ ] Fondo neutro (cartulina blanca o tela)
- [ ] Lista en papel para anotar datos de cada producto
- [ ] Suficiente espacio de almacenamiento disponible
