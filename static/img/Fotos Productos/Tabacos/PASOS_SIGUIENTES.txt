Te preparé un script para esto. Ejecutar en terminal Python:

   cd C:\Users\YASITCOMPUTER\Documents\Claude Code\e-Tabaco
   python << 'EOF'
   import pandas as pd

   # Leer los productos originales
   df = pd.read_csv('productos_export.csv')

   # Agregar columna foto_disponible (NO para todos, ya que no tienen imagen)
   df['foto_disponible'] = 'NO'

   # Reordenar columnas
   cols = ['nombre', 'marca', 'codigo', 'categoria', 'peso_gramos',
           'procedencia', 'intensidad', 'descripcion', 'precio_unidad',
           'precio_mayor', 'cantidad_minima_mayor', 'stock', 'activo',
           'imagen', 'foto_disponible']

   df = df[[c for c in cols if c in df.columns]]
   df.to_excel('catalogo-productos-v3.xlsx', index=False, sheet_name='Catalogo')
   print("✓ Excel actualizado con 226 productos")
   EOF
